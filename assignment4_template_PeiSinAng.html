Topic:

Is there an optimal time and place to fill up your vehicle in Victoria?

  • Does an area’s fuel price reflect its geographic location, or is it driven by the brands available nearby?

  • Do socioeconomically disadvantaged areas pay more or less for fuel compared to wealthier areas?

  • What is the optimal day of the week to refuel?

While browsing the ABC’s national fuel price tracker i noticed that Victoria’s data was absent, due to the way the state releases price information. As someone personally affected by the fluctuating fuel costs, I wanted to fill that gap and investigate whether there are smart times and places to refuel in Victoria, using the Servo Saver dataset.

Dataset:

  1. Servo Saver Public API

    • Source: Service Victoria

    • Content: Fuel price data for all registered service stations across Victoria

    • Last Updated: 24-hour delay released

    • Data Type: Census

    • Licence: Creative Commons Attribution 4.0 International licence

    • Limitation: Data refresh every 24-hour, there is no historical data. User will have to pull the data from the API key periodically. Coverage depends on retailer compliance.

    • Privacy/Ethics: Not personally identifiable information. Commercial data only.

  2. ABS Digital Boundary Files

    • Source: Australian Bureau of Statistics (ABS)

    • Content: Spatial boundary shapefiles used to map service station locations to statistical areas for geographic price comparison.

    • Last updated: 2021

    • Data Type: GeoPackage/Shapefile

    • Licence: Creative Commons Attribution 4.0 International licence

    • Limitation: Boundaries released in 2021 and may not reflect recent urban developments.

    • Privacy/Ethics: No privacy concerns, geographic reference data only.

  3. Socio-Economic Indexes for Areas (SEIFA), Australia 2021

    • Source: Australian Bureau of Statistics (ABS)

    • Content: Uses 2021 Census data to summarise the socio-economic characteristics of an area (such as income, education, occupation, housing and family structure)

    • Last updated: 27 April 2023 (refences 2021 Census data)

    • Data Type: Derived census-based aggregate

    • Licence: Creative Commons Attribution 4.0 International licence

    • Limitations: Based on 2021 Census data and may not reflect current socioeconomic conditions.

    • Privacy/Ethics: Aggregated area-level data, no reidentification concern.

Servo Saver API:

  1. Obtain API access and authorisation from Servo Saver Public API webpage
    • Access will take at least 24 hours, and the API key will be sent to applicants via email upon approval.
  2. Pull data
library(httr2)

# Request call 

req <- request("https://api.fuel.service.vic.gov.au/open-data/v1/fuel/prices") |> 
  req_headers(
    "User-Agent" = "MyApp/1.0",
    "x-consumer-ID" = Sys.getenv("SERVO_API_KEY"),
    "x-transactionid" = "6ff20493-4de0-4735-a036-e1e8dd41b33c"
  ) |> 
  req_user_agent("R httr2 servo-saver-client")

# Getting response 

resp <- req_perform(req) |> 
  resp_body_json(simplifyVector = TRUE) 

# Save raw data as a tibble
tibble::as_tibble(resp)
  1. Unnest raw data that has dataframes within rows (repeat for every day the data is extracted) - Data extracted at 11.50pm AEST everyday from 1 June to 5 June (Monday to Friday)

ABS Digital Boundary File and SEIFA

  1. Navigate to ABS Digital Boundary Files

  2. Download the Statistical Area Level 1 (SA1) - 2021 - Shapefile

  3. Navigate to Socio-Economic Indexes for Areas (SEIFA), Australia 2021

  4. Download the Statistical Area Level 1, Indexes, SEIFA 2021.xlsx

  5. Process and save SA1 shapefile

# Read raw shapefiles
sa1_raw <- st_read("data/SA1_2021_AUST_SHP_GDA2020/SA1_2021_AUST_GDA2020.shp")

# Filter to Victoria only
sa1_vic <- sa1_raw |> 
  filter(STE_CODE21 == "2") |> 
  select(SA1_CODE21, geometry)

saveRDS(sa1_vic, "data/sa1_vic.rds")
  1. Read, process, join and save the SEIFA SA1 data
library(readxl)
seifa_raw <- read_excel("data/Statistical Area Level 1, Indexes, SEIFA 2021.xlsx",
                        sheet = "Table 1",
                        skip = 5)
# Rename header
seifa_clean <- seifa_raw |>
  rename(
    SA1_CODE21    = 1,
    IRSD_score    = 2,
    IRSD_decile   = 3,
    IRSAD_score   = 4,
    IRSAD_decile  = 5,
    IER_score     = 6,
    IER_decile    = 7,
    IEO_score     = 8,
    IEO_decile    = 9
  ) |>
  select(SA1_CODE21, IRSD_score, IRSD_decile, IRSAD_score, IRSAD_decile,
         IER_score, IER_decile, IEO_score, IEO_decile) |>
  mutate(SA1_CODE21 = as.character(SA1_CODE21)) |>
  filter(startsWith(SA1_CODE21, "2")) #filter only Victoria SA1

# Save processed SEIFA data
saveRDS(seifa_clean, "data/seifa_clean.rds")

# Sanity check before joining 
sum(sa1_vic$SA1_CODE21 %in% seifa_clean$SA1_CODE21) 

# Join data
sa1_seifa <- sa1_vic |>
  left_join(seifa_clean, by = "SA1_CODE21")

# Save joined data 
saveRDS(sa1_seifa, "data/sa1_seifa.rds")
